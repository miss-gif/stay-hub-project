# stay-hub-project

## 구현기능

- 호텔 리스트페이지
  : 페이징 (무한 스크롤) - 완료
  : 핫딜 이벤트 (실시간 타이머) - 완료
  : 찜하기 (구독과 구독 취소)

- 호텔 상세페이지
  : 캐러셀
  : 실시간 잔여객실
  : 구글맵
  : 리뷰
  : 공유하기
  : 문서편집 (마크다운)

  ```
  npm i react-markdown
  npm install -D @tailwindcss/typography
  ```

- 찜목록
  : 드래그앤드롭

- 회원가입, 로그인
  : 소셜 로그인

- 예약
  : 예약 일정 캘린더
  : 예약 정보 폼

## 최적화 포인트

- 코드 스플리팅 - 첫 렌더링 최적화
- 대용량 리스트 최적화
- 애니메이션 최적화
- 이미지 캐러셀 최적화
- SEO 검색엔진 최적화
